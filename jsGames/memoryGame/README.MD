# Memory game
## Things I learned on this project

* DOMContentLoaded
    * With these event we wait for the _DOM_ is loaded to interact with the _DOM_ \
    [More info about DOMContentLoaded](https://developer.mozilla.org/es/docs/Web/Events/DOMContentLoaded)
* querySelector
    * This selector return the first item that matched with the specified selector. \
    [More info about querySelector](https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelector)
